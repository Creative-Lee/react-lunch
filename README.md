## 어플리케이션 실행 방법!

### 1. 패키지 다운받기

```bash
npm i
```

위 명령어로 필요한 패키지를 다운로드해 주세요!

### 2. 프로젝트 실행

```bash
npm run start
```

위 명령어를 터미널에 입력해주세요!

### 3. 등록된 음식점을 확인해보세요!

```
6개의 카테고리, 총 18개의 음식점이 등록되어 있습니다!
```
